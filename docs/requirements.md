# 开发需求文档

## 项目名称
佛山楼盘数据查看器（Foshan Property Viewer）

## 项目目标
为普通用户打造一个 Windows 桌面软件，用于查看和分析佛山市房产信息网上五个区的楼盘房源数据。

## 目标用户
- 不懂编程的普通用户
- 有购房需求、需要关注佛山楼盘价格变化的用户

## 功能需求

### 1. 数据展示
- 展示字段：面积、总价、单价、9折后单价
- 区域分类：禅城区、南海区、顺德区、三水区、高明区
- 默认排序：按单价从高到低排列
- 点击项目可查看楼栋→房源→价格的详细信息

### 2. 搜索功能
- 支持按楼盘/项目名称搜索
- 搜索结果清晰展示

### 3. 数据更新
- 手动刷新：点击按钮立即从网站抓取最新数据
- 自动采集：按设定周期（7天/1个月/季度/半年）自动抓取

### 4. 历史追踪
- 每次采集自动保存价格快照
- 可查看任意房源/楼盘的价格变化走势图
- 支持选择时间范围查看

### 5. 云端同步
- 数据存储到云端（Supabase）
- 换电脑后可登录账号恢复数据
- 本地离线也能查看最后一次同步的数据

### 6. UI/UX
- 简洁直观的界面设计
- 主色调：淡蓝色 (#4A90D9)
- 中文字体优先（微软雅黑）
- 左侧边栏导航 + 右侧内容区布局

## 非功能需求
- 运行平台：Windows 10/11
- 安装包大小：预期 60-120MB
- 离线可用：无网络时查看已缓存数据
- 数据安全：云端账号认证，数据隔离
- 采集合规：仅用于个人查询，请求频率控制

## 数据来源
- 网站：https://fsfc.fszj.foshan.gov.cn/loupan/s/
- 佛山市住房保障和房产信息网
- 通过网页数据抓取方式获取

## 版本规划
- V0.1：基础骨架 + 数据采集 + 列表展示
- V0.2：完整UI + 搜索 + 价格历史图表
- V0.3：云端同步 + 定时自动采集
- V1.0：测试完成 + 正式安装包
